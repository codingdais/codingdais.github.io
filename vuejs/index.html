# Vue

# What is vue?

Vue was created by Evan You after working for Google using AngularJs in number of projects. He later summed up his thought process: " I figured, what if i could just extract the part that i really liked about Angular and built something really lightweight" The first source code commit to the project was dated july 2013, and Vue was first released the following February, in 2014

# Why Vue.js

  • Developers can get on boarded really quickly.
  • Reactive data binding.
  • Components
  • Simple API
  • Lightweight
  • Growing Community

# Features

  • Databinding
  • Directives (v-syntax)
  • Components
  • Computed Properties
  • Watchers

# how does it work?

# Common Directives

v-if <div v-if="show"></div>
v-show <h1 v-show="ok">Hello</h1>
v-else <div v-else></div>
v-else-if <div v-else-if><div>
v-for <div v-for="item in items"></div>
v-on <a v-on:click="doSomething"></a>
v-bind <a v-bind:href="url"></a>
v-model <input  v-model="message" placeholder="edit me" />
