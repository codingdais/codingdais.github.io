# Mixins

  • Allow you to share data between different Components (stateless)
  • Better option in Vue 3 is the Composition API

# Nuxt

  • Nuxt is a universal application framework for vue.js
  • Has many of the same convertions as Vue, accept it a allows you to create Server Side Rendered (SSR) apps as well as statically generating them.
  • Big topic, will do an overview

# APIs

  • Vue.js has a number of APIs that allow us to interact with the dom. we'll take a look at some of the basic ones
  • vm.$el - the root element of the Vue instance
  • vm.$refs - Dom elements or component instances (you can access child components instance and child elements)

# Renderless Functions

  • A renderless component is a component that doesn't render any of its own HTML. Instead it only manages state and behavior, exposing a single scoped slot that gives the parent/consumer complete control over what should actually be rendered - Adam Wathan
